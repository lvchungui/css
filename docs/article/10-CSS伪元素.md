# <center>10 CSS 伪元素

- 伪元素：一般页面中的非主体内容可以使用伪元素
- 伪元素与元素的区别：
  - 元素：HTML 设置的标签
  - 伪元素：由 CSS 模拟出的标签效果


- 种类：

|  伪元素  |               作用               |
| :------: | :------------------------------: |
| ::before | 在父元素内容的最前添加一个伪元素 |
| ::after  | 在父元素内容的最后添加一个伪元素 |


- 注意点：
  - 必须设置content属性才能生效
  - 伪元素默认是行内元素
